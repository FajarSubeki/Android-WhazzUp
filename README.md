# Android-WhazzUp

WhazzUp merupakan aplikasi chatting seperti whatsapp, dimana orang satu sama lain dapat melakukan chattingan dengan teman sekontak
yang mereka miliki.

# Teknologi
1. Java
2. QuickBlox SDK
3. SuperNova Emoji

# Detail sistem / Fitur
1. Login
2. Register
3. Halaman chat dialog (Memilih dengan siapa ingin memulai chat)
4. Halaman chattingan
5. Tambah Kontak
6. Ubah nama grup
7. Tambah anggota grup
8. Keluarkan anggota
9. Realtime chat
10. Status user typing
11. Melihat jumlah pesan yang belum dibaca
12. Chat Group
13. Chat pribadi
14. Splash screen
15. Profile User
